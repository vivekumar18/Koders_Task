# Task Manager Web Application

A simple full-stack Task Manager application where users can create, update, mark as complete, and delete tasks.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js with Express
- Database: MongoDB

## Features

- Add new tasks
- View all tasks
- Update task status (Pending / Completed)
- Delete tasks
- Basic form validation
- REST API communication

## Project Structure

KODERS_TASK/
│
├── backend/
│   ├── server.js
│   ├── package.json
│   └── other backend files
│
└── frontend/
    ├── index.html
    ├── style.css
    └── script.js

## Setup Instructions

### 1. Clone the Repository
git clone <your-repository-link>
cd KODERS_TASK

How to run the project:
1. Open terminal.
2. Go to backend folder:
   cd backend
3. Install dependencies:
   npm install
4. Start the server:
   npm run dev
5. Make sure MongoDB is running.
6. Open frontend folder and open index.html in browser.
Server runs on:
http://localhost:5000
